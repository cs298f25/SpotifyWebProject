# SpotifyWebProject

Spotify Web Project.


* Create a file `.env` that contains the hostname and port number for Redis.  To Run Redis, we be using (`localhost`) and using the standard Redis port (6379). **WE NEED INFO ABOUT SPOTIFY CLIENT**

  ```
  REDIS_HOST=localhost
  REDIS_PORT=6379
  ```
